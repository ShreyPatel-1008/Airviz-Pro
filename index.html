Developer API

Access our air quality data programmatically with these simple REST endpoints:

Current AQI Data

GET /api/aqi?location=delhi

Returns current AQI and pollutant levels for the specified location.

Forecast Data

GET /api/forecast?location=delhi&days=3

Returns AQI predictions for the next 1-7 days.

Historical Trends

GET /api/trends?location=delhi&period=30

Returns historical AQI data for analysis and trend visualization.

Nearby Stations

GET /api/stations?lat=28.6139&lon=77.2090&radius=10

Returns air quality monitoring stations within specified radius (km).
